# 프로젝트

영화정보를 제공하는 웹사이트

## 기능

* 인기영화 및 모든영화의 제목, 장르, 배우등의 정보 제공                                                                                                                                                                                                                      
* 검색시 검색어가 들어간 모든영화 출력                                                                                                                                                                                                                                      
* 영화에 대한 리뷰 작성, 수정, 삭제 및 찜 구현                                                                                                                                                                                                                              
* 미디어 쿼리를 이용한 반응형 웹사이트 구현                                                                                                                                                                                                                                  
* Route를 사용한 SPA 구현
